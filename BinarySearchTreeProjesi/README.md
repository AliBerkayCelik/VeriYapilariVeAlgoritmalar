2.  5, 7'nin soluna eklenir.
3.  1, 5'in soluna eklenir.
4.  8, 7'nin sağına eklenir.
5.  3, 5'in sağına eklenir.
6.  6, 3'ün sağına eklenir.
7.  0, 1'in soluna eklenir.
8.  9, 8'in sağına eklenir.
9.  4, 1'in sağına eklenir.
